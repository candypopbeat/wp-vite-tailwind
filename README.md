## 概要
- WordpressテーマをViteとTailwindで開発するためのコード
- SassコンパイルとViteサーバー用にNode.jsが必要（無い場合はインストールしてください）

## 使い方
- Wordpressのテーマフォルダにクローンする
- Tailwindを使ってテーマ開発を進める
- 開発中はリアルタイムコンパイルを行う
- 完成後はビルド（出力）してコンパイル環境がなくても動くようにする

## コマンド
```bash
# リアルタイムコンパイル（開発中）
npm run dev

# コンパイルファイルたちの出力（完成後）
npm run build
```
